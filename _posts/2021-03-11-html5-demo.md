Note that this is a very incomplete build and basically the same as the one I posted a video of 2 years ago on YouTube. As such, there is lots of things from debugging and testing left in. Also, things like most sound effects and saving progress are missing.

I had noticeable lag when testing this on Firefox, but Chromium browsers seemed to handle it fine. Both keyboard and controller inputs should work, but make sure to set each player's control configuration in the settings accordingly.

# Controls
### Keyboard 1
WASD: Movement
P: Jump
O: Action
Q: Spawn player 2 (as player 1)/Hold to despawn self (as player 2)
Shift: Eject ability
Backspace: Pause

### Keyboard 2
Arrow Keys: Movement
M: Jump
N: Action
B: Spawn player 2 (as player 1)/Hold to despawn self (as player 2)
V: Eject ability
\ (Back Slash): Pause

### Controller
D-Pad: Movement
Bottom Button: Jump
Left Button: Action
Right Button: Spawn player 2 (as player 1)/Hold to despawn self (as player 2)
Top Button: Eject ability
Start: Pause
